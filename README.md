# Get Started 

- Environment Variables


```ts
  POSTGRES_USER=
  POSTGRES_PASSWORD=
  POSTGRES_DB=
```

- Running the dev database

```bash
  docker-compose up -d
```


- Installing dependencies

```bash
  yarn install
```

- Running the server

```bash
  yarn start:dev
```
